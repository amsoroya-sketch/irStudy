# ICRP Preparation Self-Assessment Checklist

## Purpose
Use this comprehensive checklist to:
- **Track your progress** across all skill areas
- **Identify gaps** in your preparation
- **Build confidence** as you check off completed items
- **Ensure readiness** before ICRP starts on March 2, 2026

---

## How to Use This Checklist

1. **Review monthly** - Assess yourself at the end of each preparation month
2. **Be honest** - This is for you, not for anyone else
3. **Set priorities** - Focus on unchecked items
4. **Celebrate progress** - Acknowledge what you've achieved
5. **Final check** - Complete full assessment week before ICRP starts

---

## SECTION 1: History Taking Skills

### Foundation Skills
- [ ] I can introduce myself professionally and build rapport
- [ ] I understand and can explain the 9-step history structure
- [ ] I can conduct a complete history in 8-12 minutes
- [ ] I consistently ask open-ended questions first
- [ ] I practice active listening without interrupting

### SOCRATES Pain Assessment
- [ ] I know what each letter of SOCRATES stands for
- [ ] I can assess pain systematically using SOCRATES
- [ ] I don't forget any components when assessing pain
- [ ] I can adapt SOCRATES questions based on patient responses

### History Components - Mastery
- [ ] Presenting Complaint - Can identify and document clearly
- [ ] HPC - Can take detailed chronological history
- [ ] Past Medical History - Know what to ask
- [ ] Medications - Ask about name, dose, frequency, compliance
- [ ] Allergies - Always ask, document reaction type
- [ ] Family History - Know which conditions are relevant
- [ ] Social History - Cover all aspects (SNAP: Smoking, Nutrition, Alcohol, Physical activity)
- [ ] Review of Systems - Can screen all systems efficiently
- [ ] ICE - Consistently explore Ideas, Concerns, Expectations

### Practice Volume
- [ ] Completed 50+ practice histories
- [ ] Completed 100+ practice histories
- [ ] Completed 150+ practice histories
- [ ] Completed 200+ practice histories
- [ ] Completed 250+ practice histories
- [ ] Completed 300+ practice histories
- [ ] Completed 350+ practice histories
- [ ] **Completed 400+ practice histories (GOAL)**

### System-Specific Histories (10+ each)
- [ ] Cardiovascular histories (e.g., chest pain, palpitations)
- [ ] Respiratory histories (e.g., SOB, cough)
- [ ] Gastrointestinal histories (e.g., abdominal pain, vomiting)
- [ ] Neurological histories (e.g., headache, weakness)
- [ ] Musculoskeletal histories (e.g., joint pain)

### Communication Skills
- [ ] Patient says I'm easy to talk to
- [ ] I maintain appropriate eye contact
- [ ] I use simple language, avoid jargon
- [ ] I show empathy and sensitivity
- [ ] I can handle difficult or emotional situations
- [ ] I summarize accurately at end of history

**Minimum target before ICRP:** 150+ checkmarks
