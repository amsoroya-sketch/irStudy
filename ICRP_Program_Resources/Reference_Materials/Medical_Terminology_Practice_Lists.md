# Medical Terminology Practice Lists

## Purpose
Use these lists to practice:
- **Typing:** Type these terms to improve medical vocabulary typing speed
- **Spelling:** Learn correct spelling of common medical terms
- **Recognition:** Become familiar with terms you'll encounter in ICRP
- **Pronunciation:** Practice saying these terms correctly

---

## How to Practice

### Daily Typing Practice (15-20 minutes):
1. Select one category
2. Type each term 3 times
3. Focus on accuracy over speed
4. Progress to typing definitions
5. Time yourself - aim to improve speed weekly

### Flashcard Practice:
1. Write term on one side, definition on other
2. Test yourself daily
3. Focus on terms you struggle with

## Category 9: Medical Abbreviations - Essential

### Timing & Frequency:
- **OD** / **Daily** - Once daily
- **BD** / **BID** - Twice daily
- **TDS** / **TID** - Three times daily
- **QID** - Four times daily
- **PRN** - As needed
- **Stat** - Immediately
- **Nocte** - At night
- **Mane** - In the morning

### Routes:
- **PO** - Per oral (by mouth)
- **IM** - Intramuscular
- **IV** - Intravenous
- **SC** / **SubQ** - Subcutaneous
- **PR** - Per rectum
- **PV** - Per vagina
- **SL** - Sublingual (under tongue)
- **Top** - Topical

### General:
- **Hx** - History
- **Dx** - Diagnosis
- **Tx** - Treatment
- **Rx** - Prescription
- **Sx** - Symptoms
- **PMH** - Past medical history
- **FH** - Family history
- **SH** - Social history
- **NAD** - No abnormality detected
- **WNL** - Within normal limits
- **c/o** - Complaining of
- **?** - Query / possible
