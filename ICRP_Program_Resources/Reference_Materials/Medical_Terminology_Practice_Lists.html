Medical Terminology Practice Lists

Purpose

Use these lists to practice:

  • Typing: Type these terms to improve medical vocabulary typing speed
  • Spelling: Learn correct spelling of common medical terms
  • Recognition: Become familiar with terms you'll encounter in ICRP
  • Pronunciation: Practice saying these terms correctly

How to Practice

Daily Typing Practice (15-20 minutes):

  1. Select one category
  2. Type each term 3 times
  3. Focus on accuracy over speed
  4. Progress to typing definitions
  5. Time yourself - aim to improve speed weekly

Flashcard Practice:

  1. Write term on one side, definition on other
  2. Test yourself daily
  3. Focus on terms you struggle with
