üìã How to Use These OSCE Stations

For Self-Study:

  1. Read candidate instructions thoroughly before starting
  2. Time yourself - set 8-minute timer
  3. Practice out loud - speak as if patient is present
  4. Use marking checklist to score yourself (/80)
  5. Review IMG mistakes section after each attempt
  6. Repeat until scoring consistently ‚â•70% (56/80)

For Study Partner Practice:

  1. Partner 1: Reads candidate instructions (candidate role)
  2. Partner 2: Acts as simulated patient using patient script
  3. Partner 3 (optional): Acts as examiner, times station, uses marking checklist
  4. Rotate roles each station
  5. Provide feedback using marking criteria
  6. Discuss IMG mistakes together after each station

Focus Areas by Weakness:

  • If weak on communication: Focus on O01 (breaking bad news), MX02 (patient education)
  • If weak on emergency recognition: Focus on O01 (ectopic), S01 (appendicitis)
  • If weak on examination skills: Focus on S02 (hernia examination)
  • If weak on elderly medicine: Focus on M03 (delirium), MX01 (falls)
  • If weak on structured frameworks: Focus on M01/M02 ("5 Ps"), M03 (CAM)
